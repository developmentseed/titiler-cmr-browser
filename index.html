About

This app

An interactive map showcasing titiler-cmr's ability to render NASA Earthdata assets on-the-fly. Browse datasets, adjust date ranges, and explore rendering options — all served as map tiles from a live titiler-cmr instance.

NASA CMR

The Common Metadata Repository is NASA's catalog for Earth science data. It indexes thousands of collections — satellite imagery, atmospheric data, ocean observations — and exposes a unified search API used to locate assets for rendering.

titiler-cmr

titiler-cmr is a dynamic tile server built on TiTiler that reads Cloud-Optimized GeoTIFFs and NetCDF/HDF5 assets directly from CMR, reprojects them, and serves map tiles without any preprocessing or data duplication.
